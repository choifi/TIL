# nGrinder 설치
1. 개념
nGrinder 
- 성능 테스트 도구
- Controller + Agent 구조


2. 설치 과정
# 2-1. ngrinder-3.5.9-p1-20240613 설치 

[ 빠른 시작 ]
export NGRINDER_HOME=~/dev/tools/ngrinder

java -Djava.io.tmpdir=/Users/dev/tools/nGrinder/lib \
     -jar ngrinder-controller-3.5.9-p1.war

[ 실행 ]
java -jar ngrinder-controller-3.5.9-p1.war

[ 브라우저 접속 ]
http://localhost:8080

# 2-2. Agent 연결
브라우저 접속 후 > Download Agent 클릭 > tar 파일 다운로드 됨

[ 다운로드 파일 이동 ]
mv ~/Downloads/ngrinder-agent* /Users/dev/tools/nGrinder

[ 실행 ]
cd ngrinder-agent
./run_agent.sh

[ 확인 ]
상단 메뉴 ADMIN > Agent Management에서 Agent List 확인
Agent의 status가 READY이면 성공
