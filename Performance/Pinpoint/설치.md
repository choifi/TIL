# Pinpoint 설치

## 1. 개념
- 네이버에서 개발한 APM (Application Performance Monitoring) 시스템
- 분산 시스템 환경에서 각 애플리케이션의 성능과 흐름을 시각적으로 분석하고 모니터링할 수 있는 오픈소스 솔루션 
- 요청 흐름 추적 가능

## 2. 구성 요소
- Collector : 에이전트로부터 데이터를 수신 
- Web : UI 및 시각화 
- Agent - 대상 애플리케이션에 설치하여 성능 데이터 수집 

## 3. 설치 과정
- Java 설정
- Agent attach
- 서버 실행

## 4. 문제 발생
- (Agent 연결 안됨 등)

## 5. 해결
- JVM 옵션 수정 등

## 6. 배운 점
- 요청 트레이싱 구조 이해
